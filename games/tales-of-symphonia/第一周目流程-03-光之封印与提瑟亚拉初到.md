# 第一周目流程（三）：光之封印、离开西尔瓦蓝特、提瑟亚拉初到

本页覆盖鲁因后续、阿斯卡德牧场、希玛、塔之玛娜、救赎之塔、富士山脉、梅尔托奇欧、赛巴克和梅尔托奇欧下水道。完成本篇后，第一周目会正式进入提瑟亚拉主线。

## 本篇目标

- 完成光之封印。
- 取得「西爾瓦藍特篇通關」。
- 进入提瑟亚拉并完成王都、赛巴克和下水道段。
- 为后续杰洛斯女性 NPC 对话、斗技场和弗拉诺尔事件建立地点记忆。

## 1. 鲁因后续与阿斯卡德牧场

1. 回鲁因后确认城镇状态变化，与所有可对话 NPC 对话。
2. 前往阿斯卡德牧场前补给并保存。
3. 牧场内部按房间逐一清理。敌人全部登记，宝箱全部打开。
4. 机关开启新路后，回旧区域检查遗漏。
5. Boss 战前保存，优先处理施法敌人。
6. 完成牧场剧情后，不要急着离开区域，确认地图是否还能回收宝箱。

## 2. 希玛

1. 到希玛后检查旅店、村口和关键 NPC。
2. 与所有 NPC 对话，记录剧情提示。
3. 若商店或补给点开放，登记物品。
4. 保存后前往塔之玛娜相关路线。

## 3. 塔之玛娜：光之封印

1. 进入前确认队伍状态。若塔内要求分队，确保两队都有恢复手段。
2. 每层新敌人至少击败 1 次。
3. 光线机关解开后，检查旧房间是否出现新通路或新宝箱。
4. Boss 前保存，准备恢复 TP 道具。
5. Boss 战中让治疗角色保持距离，洛伊德负责打断和牵制。
6. 完成光之封印后，按剧情前往乌玛西湖 / 托达间歇泉等地点处理后续。

## 4. 离开西尔瓦蓝特前清点

1. 回访可进入城镇，补没买的新装备和料理。
2. 检查魔物图鉴：早期世界地图、牧场、神殿、王陵和塔之玛娜的敌人是否都有记录。
3. 检查宝箱清单：临时牧场、神殿和塔之玛娜不能留空项。
4. 建立「离开西尔瓦蓝特前」永久存档。

## 5. 救赎之塔与西尔瓦蓝特基地

1. 进入救赎之塔前保存。塔内连续剧情较多，Boss 全部用 Magic Lens / 魔物观察类道具记录。
2. 按剧情推进，遇到新敌人就登记。
3. 离开塔后进入基地段。基地敌人可能阶段限定，不要逃跑。
4. 搜索支路宝箱，Boss 前保存。
5. 完成后进入提瑟亚拉，取得「西爾瓦藍特篇通關」。

## 6. 富士山脉

1. 初到提瑟亚拉后，不要急着离开山脉。沿路清敌并登记新怪物。
2. 岔路优先探索，宝箱全部开启。
3. 提瑟亚拉敌人强度提高，队伍策略改为「治疗优先」。
4. 离开山脉前保存。

## 7. 梅尔托奇欧

1. 进入王都后按区域探索：贫民区、商业区、城堡周边、可进入建筑。
2. 找到 Wonder Chef，学习本城料理。
3. 商店新物品较多，优先登记武器、防具、饰品和食材。
4. 与 NPC 对话，记录斗技场、贵族区和后续支线线索。
5. 推进主线前保存。

## 8. 提瑟亚拉桥与赛巴克

1. 过桥时留意剧情和可对话 NPC。
2. 到赛巴克后检查学院、研究设施、旅店和商店。
3. 与关键 NPC 对话推进剧情。
4. 保存后返回梅尔托奇欧相关路线。

## 9. 梅尔托奇欧下水道

1. 下水道机关涉及箱子、水路和开关。每改变一次地形，回头检查旧区域。
2. 新敌人逐个登记。
3. 宝箱全部拿完后再进入 Boss 区。
4. Boss 战前保存，装备防异常饰品。
5. 离开后再次检查梅尔托奇欧和赛巴克是否出现新对话。

## 本篇结束检查

| 项目 | 需要确认 |
| --- | --- |
| 主线 | 光之封印完成，已进入提瑟亚拉 |
| 奖杯 | 「西爾瓦藍特篇通關」已取得 |
| 怪物 | 阿斯卡德牧场、塔之玛娜、救赎之塔、富士山脉、下水道敌人已登记 |
| 宝箱 | 阿斯卡德牧场、塔之玛娜、救赎之塔、富士山脉、下水道宝箱已清 |
| 商店 | 梅尔托奇欧、赛巴克新商品已记录 |
| 存档 | 进入提瑟亚拉后保留永久存档 |
