# 第一周目流程（五）：弗拉诺尔事件、终盘与通关前清理

本页覆盖首周目后半：西尔瓦蓝特召唤精灵、暗之神殿、伊赛利亚人间牧场、尤米尔森林、拉修安溪谷、救赎之塔、弗拉诺尔事件、托伦特森林、最终区域和首周目通关前清理。首周目推荐选择柯蕾特路线。

## 本篇目标

- 完成提瑟亚拉篇和迪利斯・卡兰篇主线。
- 在弗拉诺尔事件选择柯蕾特。
- 完成能在首周目处理的支线强敌和清理项目。
- 为第 2 周目继承建立完整记录。

## 1. 西尔瓦蓝特召唤精灵与支线窗口

1. 回访高拉奇亚森林、地之神殿、特里耶特、富士山脉、鲁因、阿尔塔米拉夜晚、特里耶特遗迹、希玛、阿斯卡德和巴拉克拉夫王陵。
2. 每个旧地点只处理「新增事件、新敌人、精灵战、称号、料理、狗、商店缺口」。已完成的普通宝箱不需要重复检查。
3. 精灵或支线 Boss 全部用 Magic Lens / 魔物观察类道具记录。
4. 鲁因修复后重新检查商店和 NPC。

## 2. 暗之神殿、Linkite Tree 与伊赛利亚人间牧场

1. 在梅尔托奇欧推进暗之神殿入口剧情。
2. 进入暗之神殿前准备异常恢复和 TP 恢复道具。
3. 暗之神殿机关通常需要引导暗影类目标。每完成一次引导，检查旧区域宝箱。
4. Boss 战前保存。
5. 按剧情前往高拉奇亚森林和 Linkite Tree。若出现新敌人，全部登记。
6. 回塔之玛娜处理后续，再进入伊赛利亚人间牧场。
7. 伊赛利亚人间牧场属于重要限时区域，进入前保存，敌人、宝箱和 Boss 全部登记。

## 3. 伊赛利亚、德克家、阿尔泰莎家、赛巴克、海姆达尔

1. 回伊赛利亚后检查村内变化，与可对话 NPC 对话。
2. 到德克家推进剧情并保存。
3. 按剧情在阿尔泰莎家、赛巴克、梅尔托奇欧之间移动。每次剧情变化后重新检查赛巴克学院和梅尔托奇欧城内 NPC。
4. 阿尔塔米拉再次回访时，白天和夜间都检查一次。
5. 前往海姆达尔后，先检查商店、NPC、可调查点，再进入尤米尔森林。

## 4. 尤米尔森林与拉修安溪谷

1. 尤米尔森林以环境机关和路径切换为主。每次改变机关后，检查旧区域是否出现新路。
2. 新敌人全部登记，宝箱全部开启。
3. 森林剧情后回海姆达尔检查 NPC 变化。
4. 进入拉修安溪谷前保存。
5. 溪谷地形复杂，建议按屏幕记录路线。机关改变通路后回头检查旧路。
6. Boss 战前保存，准备恢复道具。
7. 按剧情前往 SE Abbey、梅尔托奇欧、阿尔塔米拉和 Exire，每到一处都检查商店、NPC 和女性 NPC 对话。

## 5. 救赎之塔第一次与 Welgaia

1. 进入救赎之塔前保留独立存档。
2. 塔内敌人、Boss 和连续剧情战全部登记。
3. 进入 Welgaia 后按区域探索，记录敌人、宝箱和关键道具。
4. 若出现不可返回区域，离开前检查地图。
5. 返回救赎之塔前保存。

## 6. 弗拉诺尔事件：首周目选择柯蕾特

1. 到阿尔泰莎家触发前置剧情后，前往弗拉诺尔。
2. 进入事件前建立永久存档，文件名或备注写「弗拉诺尔前」。不要覆盖。
3. 首周目目标是柯蕾特。若柯蕾特没有进入来访候选，读更早存档调整好感选项。
4. 选择柯蕾特后，取得她路线相关称号 / 道具进度。
5. 事件结束后检查名词表和称号清单，把「柯蕾特路线完成」标记清楚。
6. 之后不要覆盖弗拉诺尔前存档，该存档可用于确认路线差异或查漏。

## 7. 救赎之塔第二次、托伦特森林、最终区域

1. 救赎之塔第二次进入后，新敌人和 Boss 全部登记。
2. 完成后前往海姆达尔、托伦特森林。
3. 托伦特森林进入前保存，机关、敌人和宝箱全部处理。
4. 剧情后前往德克家、Derris-Kharlan、Welgaia 和米特斯城。
5. 最终区域按区域清图。所有新敌人都要登记。
6. 最终 Boss 前保留独立存档。若还要做支线清理，不要覆盖。

## 8. 首周目通关前清理

### 建议完成

1. 剑舞者能完成到当前可完成阶段就完成。若三战均可处理，击败第 3 战拿「刀劍舞者殺手」。
2. 魔武器支线尽量推进到可挑战阿比席恩。若等级和装备足够，首周目可完成「阿比席恩終結者」。
3. 斗技场基础挑战可做，但杰洛斯全称号和女性 NPC 线留到第 3 周目更稳。
4. 禁书三英雄可在等级足够后挑战；若困难难度压力大，可留到后续周目。
5. 料理尽量全学，熟练度可留多周目补。

### 不建议首周目强求

1. 全道具图鉴：弗拉诺尔分歧道具互斥，首周目无法完成。
2. 全魔物图鉴：克拉特斯路线和非克拉特斯路线都需要，首周目无法完成。
3. 全称号：多个角色称号依赖不同弗拉诺尔选择。
4. 100 万赌场筹码：可做，但更适合第 3 周目或第 4 周目集中处理。

## 9. 首周目通关与继承准备

1. 通关取得「提瑟亞拉篇通關」和「迪利斯・卡蘭篇通關」。若首周目全程困难，还会取得「困難突破者」。
2. 通关后进入 Grade 商店前，记录首周目缺口：怪物、道具、料理、称号、狗、宝箱、女性 NPC。
3. 继承优先级：称号、图鉴、料理熟练度、金钱 / 道具、经验或 Grade 相关项目。
4. 第 2 周目目标是克拉特斯路线，因此必须确保能继承或记录对魔物图鉴和道具图鉴有价值的项目。

## 首周目结束检查

| 项目 | 需要确认 |
| --- | --- |
| 路线 | 弗拉诺尔选择柯蕾特 |
| 主线 | 提瑟亚拉篇、迪利斯・卡兰篇完成 |
| 支线 | 魔武器已开启；剑舞者进度已记录 |
| 图鉴 | 道具、魔物缺口已写入第 2 周目清单 |
| 称号 | 柯蕾特路线相关称号已记录 |
| 存档 | 弗拉诺尔前、最终 Boss 前、通关后均有独立存档 |
