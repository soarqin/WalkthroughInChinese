# 第一周目流程（二）：帕尔玛柯斯塔、水之封印、风之封印

本页覆盖奥萨山道、伊索尔德、帕尔玛柯斯塔、帕尔玛柯斯塔牧场、托达间歇泉、阿斯卡德、鲁因和巴拉克拉夫王陵。此阶段开始出现大型城镇和牧场地图，收集压力明显增加。

## 本篇目标

- 完成水之封印与风之封印。
- 清理帕尔玛柯斯塔牧场和阿斯卡德相关剧情。
- 登记大型城镇商店、料理和可回访点。
- 继续补齐早期怪物、宝箱和 Boss 资料。

## 1. 奥萨山道

1. 从特里耶特出发后进入奥萨山道。先清道路敌人，记录新怪物。
2. 山道有支路和洞穴感路线。每次看到岔路，先走非主线方向拿宝箱。
3. 若遇到明显强敌或剧情战，先保存。打不过可以回头补给，但不要推进到无法返回的阶段后才想起。
4. 离开前确认宝箱和怪物清单。

## 2. 伊索尔德

1. 入村后先检查港口、民居、商人和关键 NPC。
2. 登记新食材和道具。
3. 推进乘船剧情前保存。
4. 若有对话选项影响好感，首周目优先柯蕾特，其次保持队伍整体自然增长，不要刻意冲杰洛斯或克拉特斯。

## 3. 帕尔玛柯斯塔：初到城市

1. 进城后不要直接推进主线。先按区域检查港口、市场、学校、总督府周边和旅店。
2. 找到 Wonder Chef，取得本城料理。
3. 商店新物品多，资金不足时按顺序购买：洛伊德武器、前排防具、治疗道具、便宜饰品、新食材。
4. 城内 NPC 对话会提供牧场和山峰线索，全部对话一次。
5. 推进主线前保存。

## 4. 哈科尼西亚山峰

1. 前往山峰前，在世界地图登记新怪物。
2. 到达后与关键 NPC 对话，确认剧情道具和返回方向。
3. 检查商人出售物。
4. 返回帕尔玛柯斯塔前保存。

## 5. 帕尔玛柯斯塔牧场

1. 进入牧场前补满恢复道具。牧场敌人和 Boss 都要登记。
2. 牧场内部按房间清理。先清敌人，再开宝箱，最后操作机关。
3. 每次打开门禁或改变机关状态后，回头检查旧区域。
4. 若剧情要求分队或潜入，先确认当前区域宝箱已经拿完。
5. Boss 战前保存。优先清小怪和施法单位，再集中攻击 Boss。
6. 完成后回帕尔玛柯斯塔，检查城内 NPC 和商店变化。

## 6. 托达码头与托达间歇泉

1. 在码头确认补给和船只剧情。若出现新商店物品，记录。
2. 进入间歇泉后，每个房间先清敌人再解谜。
3. 水路、传送或机关变化后，返回旧区域检查宝箱。
4. Boss 战前保存，保持治疗角色安全。
5. 完成水之封印后，回世界地图前检查图鉴缺口。

## 7. 阿斯卡德

1. 入城后先检查商店，登记新武器、防具、头部装备、手部装备、饰品和食材。
2. 与广场、旅店、遗迹附近 NPC 对话。
3. 推进舞台事件前保存。
4. 剧情战中记录敌人。若敌人伤害高，手动控制洛伊德贴近施法者。

## 8. 鲁因

1. 首次到达鲁因时，与所有可对话 NPC 对话。
2. 检查商店是否开放。若可买新装备，登记。
3. 鲁因后续会破坏和修复，当前阶段 NPC 状态不要跳过。
4. 保存后准备前往巴拉克拉夫王陵。

## 9. 巴拉克拉夫王陵：风之封印

1. 进入前准备解异常状态道具。
2. 王陵机关以风向、石块或通路变化为主。每次机关变化后回头检查。
3. 偏离主路的房间优先搜索，宝箱不能漏。
4. 新敌人全部登记。
5. Boss 战前保存。若 Boss 使用范围攻击，队伍分散站位，洛伊德负责牵制。
6. 完成风之封印后，回阿斯卡德 / 鲁因推进剧情。

## 本篇结束检查

| 项目 | 需要确认 |
| --- | --- |
| 料理 | 帕尔玛柯斯塔料理已学习 |
| 怪物 | 奥萨山道、帕尔玛柯斯塔牧场、托达间歇泉、阿斯卡德周边、王陵敌人已登记 |
| 宝箱 | 牧场、间歇泉、王陵宝箱已清 |
| 商店 | 伊索尔德、帕尔玛柯斯塔、阿斯卡德、鲁因商店缺口已记录 |
| 剧情 | 水之封印、风之封印完成 |
| 存档 | 风之封印完成后保留永久存档 |
