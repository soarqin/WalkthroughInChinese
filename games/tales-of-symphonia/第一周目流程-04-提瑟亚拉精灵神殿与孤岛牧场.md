# 第一周目流程（四）：提瑟亚拉精灵神殿与孤岛牧场

本页覆盖水穗、奥赛特、托伊兹谷矿山、雷之神殿、地之神殿、弗拉诺尔、冰之神殿、阿尔塔米拉和孤岛人间牧场。此阶段开始密集出现后期支线入口，尤其是魔武器、赌场、杰洛斯女性 NPC 和弗拉诺尔事件前置。

## 本篇目标

- 完成雷、地、冰相关主线迷宫。
- 在弗拉诺尔开启魔武器支线。
- 登记阿尔塔米拉、赌场和夜间区域。
- 完成孤岛人间牧场。

## 1. 高拉奇亚森林

1. 进入森林后按屏幕清图，先清敌人，再找宝箱。
2. 森林道路容易绕回，建议固定按左手或右手方向探索。
3. 新敌人全部登记。
4. 通过后前往水穗。

## 2. 水穗

1. 与村内所有 NPC 对话，尤其是主线人物。
2. 检查商店、补给和可调查点。
3. 若出现影响椎名或队伍好感的对话，首周目仍以柯蕾特路线为优先，不刻意追其他角色。
4. 推进剧情后保存。

## 3. 奥赛特与阿尔泰莎家

1. 到奥赛特后先检查村内商店和 NPC。
2. 前往阿尔泰莎家，完成普蕾榭雅相关剧情。
3. 矿山前补给 Life Bottle、Orange Gel 和异常恢复道具。
4. 保存后进入托伊兹谷矿山。

## 4. 托伊兹谷矿山

1. 矿山机关与矿车 / 轨道路线相关。每次改变路线后，检查新旧两边。
2. 所有新敌人登记。
3. 看到宝箱先拿，不要等主线完成后补。
4. Boss 战前保存。困难难度下优先清小怪，减少治疗压力。
5. 完成后回奥赛特、水穗和世界地图推进剧情。

## 5. 雷之神殿

1. 进入前准备防雷或提升生存的装备。
2. 雷机关会改变通路，操作一次就回头检查一次。
3. 新敌人全部登记。
4. Boss 战前保存。若 Boss 法术压制严重，手动控制洛伊德优先打断。
5. 完成后进入提瑟亚拉基地段。

## 6. 提瑟亚拉基地

1. 基地敌人可能阶段限定，遇到就打。
2. 搜索所有房间和支路。
3. 机关开启后回头确认宝箱。
4. 剧情战前保存。

## 7. 地之神殿

1. 前往神殿前在世界地图补新怪物。
2. 神殿内按区域推进，不要绕过支路。
3. 若有特殊 NPC 或精灵相关机制，先完成对话再操作机关。
4. Boss 战前保存，准备 TP 恢复道具。
5. 完成后确认是否解锁新的召唤或精灵支线。

## 8. 弗拉诺尔：第一次重要回访点

1. 到弗拉诺尔前，在雪原击败 Ice Spider、Penguinist 等敌人。
2. Penguinist 会掉落 Penguinist Quill。进入冰之神殿前准备 3 个。
3. 进城后先命名入口附近的狗。
4. 与 Abyssion 对话，开启 Devil's Arms / 魔武器支线。此支线后期对应「阿比席恩終結者」。
5. 与询问 Celsius' Tear 的 NPC 对话，确认 Penguinist Quill 需求。
6. 进入武器 / 防具店，购买限时 Elder Cloak。资金不足也应优先买下。
7. 到饰品店与老人对话，交付材料取得 Penguinist Gloves。
8. 前往教会，在后方角落找到 Wonder Chef，学习 Quiche。
9. 到东侧区域命名第二只狗，购买 Flanoir Potion 和食材。
10. 记住医生办公室位置，后续弗拉诺尔好感事件会回到这里。

## 9. 冰之神殿

1. 进入前确认 3 个 Penguinist Quill 相关事件已处理。
2. 滑冰、推箱、冰路机关完成后都要回头检查宝箱。
3. 新敌人全部登记。
4. Boss 战前保存并准备防冰装备。
5. 完成后返回奥赛特 / 阿尔泰莎家推进剧情。

## 10. 阿尔塔米拉

1. 白天先检查酒店、游乐设施、商店、海边和赌场入口。
2. 若可切换夜晚，夜间 NPC、酒店楼层、游乐设施和海边重新检查。
3. 记录赌场位置。首周目不强制刷 100 万筹码，可留到第 3 周目或第 4 周目。
4. 由于杰洛斯此时在队，女性 NPC 对话可以开始顺路做，但首周目不以「傑洛斯・泡妞高手」为最终目标，漏项留到第 3 周目专门处理。

## 11. 异界之门、帕尔玛柯斯塔回访、孤岛人间牧场

1. 按剧情前往 Otherworldly Gate。
2. 回帕尔玛柯斯塔后，检查商店、NPC 和女性 NPC 对话。
3. 进入孤岛人间牧场前补给并保存。
4. 牧场内部按房间清理，敌人和宝箱全部登记。
5. 若地图有传送、开关或门禁，先列清区域，再推进主线。
6. Boss 战前另存。完成后返回阿尔塔米拉确认剧情变化。

## 本篇结束检查

| 项目 | 需要确认 |
| --- | --- |
| 主线 | 雷、地、冰相关主线迷宫和孤岛牧场完成 |
| 支线 | 魔武器支线已从弗拉诺尔开启 |
| 料理 | Quiche 和本阶段城镇料理已学习 |
| 狗 | 弗拉诺尔两只狗已命名 |
| 怪物 | 高拉奇亚森林、矿山、雷之神殿、地之神殿、雪原、冰之神殿、孤岛牧场敌人已登记 |
| 商店 | Elder Cloak 已购买；阿尔塔米拉和弗拉诺尔商店已记录 |
| 存档 | 孤岛牧场结束后保留永久存档 |
