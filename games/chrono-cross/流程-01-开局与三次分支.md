# 开局与三次分支

## Introduction

游戏中存在三次重大选择，会决定当前周目中哪些角色可以加入。

本攻略假设你在 Branch 3 中选择了 「Save Kid」。如果不救 Kid，你会在较晚的时间点获得 Astral Amulet，这会打乱本攻略的顺序。

**Branch 1：与 Kid 同行**
*   接受：Kid
*   拒绝：Leena、Kid
*   触发时机：第一次访问 Another World 的 Cape Howl。

**Branch 2：选择向导**
*   Guile
*   Pierre
*   Nikki
*   触发时机：在 Termina 与擦拭雕像的男人交谈后。

**Branch 3：救 Kid 还是当个混蛋**
*   Save Kid：Korcha、Razzly、Mel、Kid
*   Be an asshole：Glenn、Macha、Doc、Kid
*   触发时机：Viper Manor 事件后。

**关于全收集**
如果你想用最少的周目达成全收集，第一周目建议正常游玩并确保拿到分支限定角色和 Branch 3 的 Frame。第二周目在 Harle 还在队伍中时达成任意结局。第三周目完成全收集。
Harle 只有在第三周目接近尾声且正确使用 Chrono Cross 时才会成为永久成员。

### Branch 3 - 拯救 Kid
触发时机：Viper Manor 事件后。

**Save Kid**
当 Korcha 询问该怎么办时，选择 「Find a way to save her」。

*   **加入 (Mel)**：获得船后，带着 Kid 再次访问 Another World 的 Guldove。

**Be an asshole**

*   **加入 (Glenn)**：回到 Another World 的 Termina 后不久加入。
*   **加入 (Macha)**：在 Another World 的 Termina 招募 Glenn 后加入。
