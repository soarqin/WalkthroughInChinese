# 流程 02：Water Dragon Isle 至 The Dead Sea

本章覆盖从获得船到进入 Dead Sea 的完整流程，包括 Water Dragon Isle、Mount Pyre、Fort Dragonia、Temporal Vortex、S.S. Zelbess、Isle of the Damned 和 Dead Sea。

## Another World · Hermit's Hideaway

获得船后，前往 Hermit's Hideaway（Isle of the Damned 左边的小岛）。

到达后 Harle 会出现并挑战你。

### Boss：Harle（黑）

- HP：400 / 攻击：32 / 防御：10 / 魔防：5
- 偷窃：PhotonBeam（普通）/ MoonGlasses（稀有）
- 掉落：PhotonBeam / MoonGlasses
- 金币：564G

务必将 MoonGlasses 作为稀有掉落或稀有偷窃获得——之后很有用。战斗后跟随 Radius 进入地下密室对话。

## S.S. Invincible（Another World）

位于 Hermit's Hideaway 南边的迷雾区域。登船后 Fargo 和海盗们会迎接你。

### 第一场遭遇战

如果选择了 Save Kid 分支：这三场战斗可以逃跑跳过，不会获得 Star。如果选择了 Leave Kid Alone 分支：每场都是 Boss 战，各给一颗 ★。

[![截图](./assets/steam-081.webp)](./assets/steam-081.webp)

**Boss：Polly（红 HP540）**——偷窃 Sight Scope / Inferno，掉落 MagmaBurst
**Boss：Fargo（蓝 HP660）**——偷窃 Capsule / PhotonBeam，掉落 Meteorite / PhotonBeam

使用红元素攻击。如果队伍中有 Glenn，可以尝试 Serge 的 Dash&Slash + Glenn 的 Dash&Gash 双人技 X-Strike。

### 幽灵船探索

醒来发现被关在船舱。

1. 击败骷髅。不死系敌人用白元素或 Recovery 元素。
2. 与每个海盗对话三次获取钥匙信息。
3. 向西走进入第一个门，宝箱有 **AquaBall**（蓝元素）。
4. 走廊尽头捡起 **FirePillar**（红元素），爬梯子上楼。
5. 向东走，其中一个房间消灭隐藏 Wraith 后获得 **Panacea**。
6. 爬通风管道到另一边，获得 **Capsule**。与前端海盗对话获得钥匙。
7. 用钥匙打开船舱附近的门。二楼左侧第二个房间与海盗对话获得 **@Iron**。
8. 经过炮口到船舷，然后回到船内。

### Pip 招募

在满是怪物的房间里看到 Pip。与他交谈后他会逃跑，追到房间东南角围栏处再次对话，**Pip 加入**。

Pip 是独特角色：多使用红、黄、黑元素会偏向恶魔方向进化，多使用绿、蓝、白元素会偏向天使方向。

## Home World · Water Dragon Isle

与 Arni Village 栈桥的渔夫对话，付 100G 乘船前往。

1. 用睡莲叶过河，宝箱中有 **AquaBall**、**IceBlast**、**@Iron**。

## Another World · Mount Pyre

位于主岛东侧。进入前准备充足的 Capsule 和 Ointment。

1. 进入后先不要冻结岩浆，冲刺到西北方向宝箱获得 **MagmaBurst**。冲回安全地带后冻结岩浆。
2. 向北走，击败 Cat Burglar 获得 **Tablet**。
3. 向东北走到出口，遇到 Shaker Brothers。Solt 会殴打 Peppor，优先攻击 Solt 直到他停手，再用元素攻击两人。
4. 继续向东北走到下一区域，使用 Ice Breath 冻结岩浆，向南滑下获得 **MagmaBomb**。
5. 滑下斜坡回到第一画面。向北走到有岩浆的区域，在冻结前冲刺到左边获得 **↓Inferno**。返回后冻结岩浆。
6. 右侧楼梯上有 **Capsule**，下楼右侧有 **Tablet**。
7. 转身上几个梯子，见到记录点存档。继续前进，小火龙挑战你。

### Boss：Fire Dragon（红 HP850）

- 偷窃：Magic Ring（普通）/ Wisp Cape（稀有）
- 掉落：Weaken / Magic Ring
- 金币：730G

不要让场地变成全红。保持 HP 在 125 以上。尽量让 Kid 偷窃 Magic Ring。

龙被击败后离开。存档后继续前往出口，Dragoon Devas 拦住你。

### Boss：Karsh（绿 HP370）、Zoah（黄 HP500）、Marcy（蓝 HP300）

这是使用 FrogPrince 召唤的最佳时机。确保其他角色在元素槽 Lv.1 位置装备至少两个蓝元素。成功释放召唤几乎可以秒杀三人。
如果不用召唤，优先解决 Marcy（她会用 IceBlast 冻结），然后用绿元素攻击 Zoah，最后 Karsh。

## Another World · Fort Dragonia（临界点）

**警告**：完成 Fort Dragonia 后，部分区域和所有同伴将暂时不可用。这是游戏的关键转折点。

### 四个水晶解谜

#### 红水晶（右下）

龙像会根据你选择的方向掉落——龙像朝向的方向视为「北」。目标是掉到房间西侧的洞里关闭水晶。

| 方向 | 区域 |
|---|---|
| 北 | 红色治疗金字塔 |
| 南 | Volcano Trap、Flame Charm、FirePillar、MagmaBurst |
| 东 | 与 Cybots 战斗 |
| 西 | 红水晶柱 |

[![截图](./assets/steam-085.webp)](./assets/steam-085.webp)

#### 黄水晶（左下）

圆形双层迷宫。到西侧激活机器，将队员分配为 Head、Body、Tail。回到主厅使用新桥激活蓝色按钮。按 Select 调整队伍顺序可改变迷宫布局：

| 顺序 | 效果 |
|---|---|
| Head-Body-Tail | 激活蓝色开关降下楼梯 |
| Body-Tail-Head | 获得 Earth Charm |
| Tail-Head-Body | 获得 ElectroBolt |
| Tail-Body-Head | 降下水晶平台 |

#### 绿水晶（右上）

迷宫区域。宝箱有 HealAll、Bushbasher、Tornado Trap、Antidote、AeroBlaster。

快速路线：第一个左转下楼梯右转，沿路走，分岔路上方走，上楼梯继续，踩蓝色按钮降下捷径。

### Boss：Taurusoid（绿 HP1200）

- 偷窃：↓Tornado / ↓Sonja
- 掉落：@Fang / Iron Vest
- 金币：1076G

**不要装备黄元素**——使用黄元素会触发 Taurusoid 释放 HealAll，大幅延长战斗。

#### 蓝水晶（左上）

三对宝箱通过开关控制桥的旋转。从左边开始：第一对（关、开），第二对（关、开），第三对（关、开）。一个出口通向 Sea Charm，另一个通向 Dragoon 敌人。

### Boss：Giant Gloop（蓝 HP800）

- 偷窃：Antiviral Cap
- 掉落：AquaBall / IceBlast
- 金币：800G

设置 IceBerg 陷阱，让 Kid 偷窃 Antiviral Cap。全员装备 Medicine 解冻。使用 Lv.4 红元素如 MagmaBurst。

### 关闭所有水晶后

返回主厅，使用记录点，乘电梯上去。

### Boss：SunOfAGun（白 HP365）

- 偷窃：@Screw
- 掉落：@Iron
- 金币：840G

SunOfAGun 使用 ModeChange 切换表情和攻击模式：
- 笑脸：攻击 -> Kissy-Wissy + ModeChange
- 空白：AntiBlack -> HolyLight + ModeChange
- 悲伤：WeakMinded -> StrongMinded + ModeChange
- 愤怒：PhotonBeam + MeteorShower + ModeChange

多用黑魔法攻击，保持场地黑色。对 Serge 使用 TurnBlack 防护白魔法。设置 HolyLight 陷阱。

关闭最后一个水晶，乘电梯上去。这里就是游戏开篇的迷宫。向右走到岔路口，东南方向获得 **Capsule**。探索其余区域可获得 **Daemon Charm**、**Angel Charm**、**MeteorShower**。

### Boss：Bunyip

第一形态（红 HP400）用蓝元素攻击。第二形态（黑 HP1200）用 Serge 物理攻击。设置 Inferno 和 FreeFall 陷阱。可偷窃 FreeFall。

### Boss：Viper（黄 HP820）

- 偷窃：Stamina Ring
- 掉落：Dragoon Gauntlet / Mythril Mail

用最强元素和 Tech 技能攻击，偷取 Stamina Ring。

### Boss：Lynx（黑 HP1000）

- 偷窃：Pendragon Sigil B / Sky Djinn Ring
- 掉落：Pendragon Sigil B / Sky Djinn Ring
- 金币：666G

比第一次遭遇更强。两名队员用白元素，Serge 物理攻击。对队友使用 TurnWhite 提升输出。

成就「The Nemesis」：在 Fort Dragonia 击败 Lynx。

过场后 Serge 的灵魂被转移到 Lynx 体内，你将控制 Lynx 与自己的朋友战斗——这场战斗无法真正获胜。

## Temporal Vortex

Fort Dragonia 事件后，你独自一人在一个色彩奇异的地方。

1. 沿树向上走，到山顶把巨石推下。追踪巨石所在位置再推一次。绕树获得 **Revive**（白元素）。
2. 走右侧树枝让果实掉落。Sprigg 会出来捡果实，趁门开着溜进她家。过夜后尝试离开，**Sprigg 加入**。
3. Sprigg 可以复制被她用物理攻击击败的怪物形态和能力。
4. 进入顶部房子，Serge 的幻影出现。跟着他进入同侧的门，重复三次。Harle 怜悯你并 **加入**。
5. 探索塔获得 **@Mythril** 和 **Pendragon Sigil C**，穿过洞口到达 **Home World Hydra Marshes**。

## 获取 Einlanzer（Home World）

### Death's Door

前往 Dead Sea（世界地图东南方向），在巨大雾气底部使用「Fiddler Crab」进入。没有 Einlanzer 无法继续前进。

### Hermit Hideaway（Home）

前往 Hermit Hideaway。如果之前在 Another 使用了 Ice Breath，这里会看到一株植物——这是 Turnip，但现在还拔不出来（需要 Poshul）。

Radius 离开后带回 **Garai's Keepsake**。自动离开区域后重新进入，爬上树顶获得 **Energizer Suit**。树下房子左侧宝箱有 **Resistance Ring**。

### Isle of the Damned（Home）

1. 向北走，击败 Will O'Wisp 让它爆炸烧掉黑门。
2. 引诱另一个 Will O'Wisp 到北面的黑门，爆炸后进入获得 **Earthquake**（黄元素）。
3. 东南方向的门同样方式打开，获得 **@Carapace**。
4. 进入第一个烧掉的门。大多数敌人是红或黑色。
5. 到巨大骨架下方进入，向南走传送到洞穴左侧。
6. 向上走到镜子处，使用 **Garai Keepsake** 通过。
7. 引诱 Will O'Wisp 绕区域（上-左-下-右），爬骨梯下去，在黑墙旁与 Wisp 战斗获得 **Inferno**。
8. 引诱 Wisp 到盔甲处烧掉，与 Wight Knight 战斗。务必让 Sprigg Dopplegang 这个一次性敌人。掉落 **Golden Tiara**。
9. 进入中间的黄色门，Lynx 必须在领队位置。阅读墓碑开始 Boss 战。

### Boss：Garai（白 HP1987）

- 掉落：Dragoon's Honor
- 金币：1358G

Garai 攻击非常残暴。让 Harle 使用 MoonShine 保护队伍，用最强黑元素（如 FreeFall）攻击。Lynx 使用 Tech 技能。对 Garai 使用 Weaken，对 Lynx 使用 Strengthen。

胜利后获得 **Einlanzer**。重新阅读墓碑看新铭文。将 Einlanzer 带到 Death's Door 中和 Masamune，进入 Dead Sea。

## The Dead Sea

有自己的世界地图。

### 世界地图

1. 向北走获得 **Anti-Red**。
2. 向东进入 Highway Ruins。

### Highway Ruins

1. 沿楼梯旁向北走到下一画面，爬楼梯后在桥下东南方获得 **@Screw**。
2. 向东北走，梯子上小房间有 **@Screw**。
3. 右下楼梯向上获得 **@Mythril**。向南到下一区域。
4. 向西南走获得 **FreeFall**，之后会从桥上掉落。
5. 爬回来向右下获得 **Capsule**。返回向北走遇到 Highwayman。

### Boss：Highwayman（蓝 HP2000）

- 掉落：Nimble / Mythril
- 金币：1100G

对 Lynx 使用 TurnWhite 后攻击。Highwayman 使用 ExhaustGas 全体致盲，用 Purify、Panacea 或 Blackout 解除。Rampage 伤害很高，立即回复。

### City Ruins

1. 进入后右侧获得 **Resistance Belt**。
2. 向东南走进入建筑获得 **Kung-fu Shoes**。
3. 返回向西走，爬梯子到房间按红色按钮，系统故障强制离开。
4. 向西南获得 **Earring of Light**。
5. 向南获得 **Stamina Ring**。继续向南出到 Dead Sea 地图。
6. 南-西-南获得 **Anti-Green**。向东南进入 Tower of Geddon。

### Tower of Geddon

1. 爬两层楼梯向西获得 **@Feather**。
2. 向东到火车站，与长凳上的 Geos 交谈，他会消失。火车顶部有 **Purify**。
3. 检查 Geos 消失处获得 **Station Pass**。通过检票口。
4. 右侧宝箱有 **HolyLight**。进入西侧房间，底部有 **Medical Book**（用于 Doc 的 Lv.7 必杀技）。
5. 滑下通风管道回到大厅，上楼梯到顶层。
6. 最左侧按红色按钮，让电梯下到 1F。跟随电梯获得 **Trashy Tiara**。
7. 回三楼左转出电梯。向右走看到幽灵，再往里有笑声，一个东西出现然后走向幽灵女孩。
8. 右侧宝箱有 **HealAll**。左侧宝箱有 **CurePlus**。
9. 上绿色舞台获得 **Prop Sword**（用于 Pierre 的终极装备）。

[![截图](./assets/steam-124.webp)](./assets/steam-124.webp)

[![截图](./assets/steam-125.webp)](./assets/steam-125.webp)

[![截图](./assets/steam-126.webp)](./assets/steam-126.webp)

10. 看到孩童 Lucca 的幽灵跑过通道。爬梯子过通道获得 **SealAll**。
11. 继续探索：**@Mythril**、**Diminish**、**Screwy Dress**、**HellBound**。

> **注意**：与 Nadia's Bell 互动后将无法再进入 Dead Sea。在此之前确保收集完所有物品和敌人！

### Boss：Miguel（白 HP1950）

- 偷窃：Energizer Suit
- 掉落：Dreamer's Scarf
- 金币：1540G

Miguel 攻击很强。对 Lynx 使用 TurnWhite。装备大量回复元素。攻击并用最强黑元素。Miguel 使用 Anti-Black，用 Purify 或 BlackOut 解除。使用 WeakMinded 后接 Lynx 的 Tech 技能。
