# 快速白金 / 全成就路线

Steam 版没有白金奖杯；本页的「白金」指全成就。路线参考 `~/quick-trophies.md` 整理，并将其中的非官方中文称呼尽量还原为游戏内英文名词。

本页不是逐步流程攻略，而是面向 `Continue+` / `New Game+` 的最短收集规划。第一次游玩请同时对照完整流程页，尤其是角色、`Frame` 和 `Level 7 Tech` 的触发条件。

## 核心目标

- 一周目先留下关键存档，减少二周目与三周目的重复流程。
- 角色收集依赖三次分支：`Leena` / `Kid`、`Guile` / `Pierre` / `Nikki`、`Save Kid` / `Don't save Kid`。
- `Serge`、`Leena`、`Razzly` 的三人 `Tech` 是长线风险点；务必按本页路线保留 `Leena` 与 `Razzly` 的条件。
- `Harle` 离队前要留一个可打最终 Boss 的存档，用于补角色记录。
- 最终用取得 `Chrono Cross` 的通关记录强化存档，再在 `Temporal Vortex` 召回历代记录中的队友。

## 存档点总览

| 存档 | 时间点 | 用途 |

| C | `Viper Manor` 后、在 `Guldove` 选择是否救 `Kid` 前 | 分流 `Save Kid` / `Don't save Kid` |
| D | `Save Kid` 路线，救 `Razzly` 前 | `Ending 4`；确认先救 `Razzly` 再打 `Hydra` |

| F | 读取 C 后走 `Don't save Kid`，完成 `S.S. Invincible` 并收齐 `Glenn`、`Macha`、`Doc` | 分支角色记录 |
| G | 变成 `Lynx` 后，可在大地图行动时 | `Ending 5`；`Harle` 仍在队伍时的角色记录 |
| H | `Another World` 的 `Viper Manor` 救出 `Riddel` 后 | `Ending 6` |
| I | `Harle` 离队后、击败 `FATE` 前，尚未持有 `Mastermune` | `Ending 8` |

## 一周目指定路线

1. 刚离开 `Arni Village` 后立刻保存为存档 A。
2. 在 `Opassa Beach` 与 `Leena` 对话时选择 `I Remember`，再选择 `We'll never forget this day`。之后在 `Cape Howl` 连续拒绝 `Kid` 三次，以招募 `Leena`。
3. 到 `Termina` 选择向导前保存为存档 B。
4. 首周目建议走 `Nikki` 线。进入 `Shadow Forest` 时，先用 `Aroma Pouch` 引走红色 `Bulb`，让它吃掉挡路植物并取得 `Skullduggery` `Frame`。
5. `Viper Manor` 事件后，在 `Guldove` 选择是否救 `Kid` 前保存为存档 C。
6. 选择 `Save Kid`。进入 `Hydra Marshes` 后，先救出 `Razzly`，并在救她前保存为存档 D。打 `Hydra` 时不要让 `Razzly` 入队，这是后续 `Razzly` `Level 7 Tech` 的关键条件。
7. `Kid` 重新加入后，确认 `Nikki` 和 `Razzly` 都已在角色记录中。进入 `Mount Pyre` 前保存为存档 E。之后前往 `Water Dragon Isle`，触发矮人袭击事件，并确保 `Rosetta` 死亡，以保留 `Razzly` 后续 `Level 7 Tech` 条件。
8. 读取存档 C，选择 `Don't save Kid`。完成 `S.S. Invincible` 事件并收集 `Glenn`、`Macha`、`Doc`，保存为存档 F。`Macha` 会给 `Tropical Paradise` `Frame`，顺手记录。
9. 变成 `Lynx` 后，在大地图保存为存档 G。此时 `Harle` 仍在队伍，适合补早期结局与角色记录。
10. 救出 `Riddel` 后保存为存档 H。此前注意两件事：变成猫时在 `S.S. Zelbess` 的厨房与猫对话拿 `Monster Mouth` `Frame`；去 `The Dead Sea` 前，在 `Another World` 的 `Marbule` 取得 `Valencian Cloth` `Frame`，否则 `Black Dragon` 醒来后会错过。
11. 完成 `Marbule` 复兴前置，让 `Nikki` 找 `Sage` 学歌。`The Dead Sea` 后优先在 `S.S. Zelbess` 举办 `Magical Dreamers` 演唱会。
12. `Harle` 离队后，在击败 `FATE` 前保存两份：尚未持有 `Mastermune` 的存档 I，以及击败 `Dario` 并取得 `Mastermune` 后的存档 J。
13. `Terra Tower` 出现后保存为存档 K。
14. 救 `Kid` 时，在最初的 `Clock-like machine` 场景输入 `L1, Triangle, L1, Triangle`，取得 `Kid` 的 `Hot Shot`。通关前保留一个已经取得 `Chrono Cross` 的存档 L。

## 结局回收顺序

读取对应存档，在 `Opassa Beach` 使用 `Time Egg`，直接击败 `Time Devourer`。

| 读取存档 | 目标 |
| --- | --- |
| A | `Ending 1: The Developers' Room` |
| B | `Ending 2: The Acacian Empire` |
| D | `Ending 4: The Shop Staff and the Three Unlikely Heroes` |
| E | `Ending 3: The Magical Dreamers` |
| G | `Ending 5: The Rebirth of Marbule` |
| H | `Ending 6: The Last Stand` |
| I | `Ending 8: The Dark Fate` |
| J | `Ending 7: The Viper Orphanage` |
| K | `Ending 9: The Dragon God's Revenge` |

`Good Ending` 需要在最终战中按 `Yellow, Red, Green, Blue, Black, White` 的顺序铺满 `Element`，最后使用 `Chrono Cross`。

## 角色收集顺序

1. 读取存档 B，分别补 `Guile` / `Pierre` / `Nikki` 的三选一路线；招募后即可击败最终 Boss，把角色记录写入通关记录。
2. 读取存档 F，`Don't save Kid` 路线的 `Glenn`、`Macha`、`Doc` 很快能收齐，之后直接击败最终 Boss。
3. 读取存档 G 或 H，在 `Harle` 仍在队伍时击败最终 Boss，把 `Harle` 写入记录。
4. 其他角色按完整流程页补齐。除非主动选错拒绝入队，大多不会错过。
5. 用最后覆盖过的通关记录强化存档 L。前往 `Home World` 的 `Hydra Marshes`，使用 `Astral Amulet` 回到 `Temporal Vortex`。在 `Sprigg` 家附近有提示的位置使用 `Chrono Cross`，即可召回过去通关记录中曾招募过的全部角色。

## `Level 7 Tech` 风险点

- `Kid`：救 `Kid` 的记忆段落中，在第一个 `Clock-like machine` 场景输入 `L1, Triangle, L1, Triangle`，取得 `Hot Shot`。
- `Razzly`：必须先救 `Razzly`，但打 `Hydra` 时不要带她；第一次去 `Water Dragon Isle` 时要确保 `Rosetta` 死亡。`Terra Tower` 出现后，带 `Razzly` 去 `Home World` 的 `Water Dragon Isle`，在姐姐墓前取得 `Raz-Flower`。
- `Leena`：前期 `Opassa Beach` 对话必须选择 `I Remember` 与 `We'll never forget this day`；变回 `Serge` 后，带她回 `Home World` 的 `Arni Village` 找祖母取得 `Maiden's Faith`。
- `Serge` / `Leena` / `Razzly` 三人 `Tech`：路线必须同时满足 `Leena` 与 `Razzly` 条件，后期再确认三人都在队伍记录中。
- `Sprigg`：`Continue+` 后进入 `Bend of Time` 的里层门，与 `Land Octopus` 对话后绕场跑三圈触发战斗。装备 `Forget-me-not Pot`，让装备者或 `Sprigg` 击杀 `Ozzie, Flea, and Slash` 中最右侧的 `Slash`，取得对应变身记录。`Forget-me-not Pot` 最早可在 `Chronopolis` 取得。
- `Lynx` 与 `Harle`：满足星级条件后可直接在 `Continue+` 存档中使用。

## `Rainbow Gear`

1. 确保已经收集足够 `Rainbow Shell`。
2. 在复兴后的 `Marbule`，到 `Black Dragon` 所在洞穴的最左侧 `Demi-human NPC` 处购买 `Master Hammer`。
3. 让队伍中的 `Zappa` 与 `Another World` 的 `Zappa` 对话，解锁 `Rainbow Gear` 锻造。
4. `Rainbow Gear` 需要 `Shiny` 系材料。用 `Summon Element` 击杀怪物才能稳定取得这些材料；`Summon Element` 通常需要 8 星级以上，建议打完 `Terra Tower`、接近 48 星级后再集中刷。`Terra Tower` 内部 Boss 固定掉落 `Field Elements`，便于铺场后召唤。

## 小游戏成就

- `Dragon Feeding`：初次潜入 `Viper Manor` 时即可挑战，喂满 100 只龙。
- `Grand Slam`：后期拿到 `Forget-me-not Pot` 并收集强力怪物后再回来打，容错更高。
- `Compass Roulette`：落到 `S` 会失败。按住确认键时指针会跳到下一个方位，例如 `W` 跳 `N`、`N` 跳 `E`。通常在指针从 `E` 快到 `S` 时，听到投币声再确认，可让它转 2 到 3 个方位，较容易落在 `W` 或 `N`。

## 收尾提醒

- 本路线最容易出错的是 `Leena` 前期对话、`Razzly` 的救援 / `Hydra` 战 / `Rosetta` 死亡条件，以及 `Kid` 的 `Hot Shot` 输入。
- 严格按存档点操作后，剩余重复工作主要是 500 次击杀、99 星级、`Rainbow Gear` 材料和零散 `Frame` / `Tech` 清单。
- `For all the Dreamers!` 相关内容请在完成 `Chrono Cross` 与 `Radical Dreamers` 主路线后再统一清理。
