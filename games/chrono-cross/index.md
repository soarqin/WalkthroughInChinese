# CHRONO CROSS: THE RADICAL DREAMERS EDITION 完成攻略

本攻略整理自 Steam Community Guide「Chrono Cross Completion」，面向 `CHRONO CROSS: THE RADICAL DREAMERS EDITION`。内容覆盖 `Chrono Cross` 的角色招募、`Level 7 Tech`、`Frame`、结局、`Rainbow Shell`、可选成就、可选 Boss，以及 `Radical Dreamers` 的路线和地图提示。

由于本作没有官方中文版，攻略中所有游戏内英文名词均保留原文，包括角色名、地名、道具名、`Frame`、`Tech`、结局名、菜单选项和世界名。

## 攻略范围

- 目标：尽量少周目完成角色、`Tech`、`Frame`、结局、`Rainbow Shell`、可选成就与关键收集。
- 适用版本：Steam 版 `CHRONO CROSS: THE RADICAL DREAMERS EDITION`；其他平台同版内容可参考。
- 核心分歧：`Branch 1` 是否与 `Kid` 同行、`Branch 2` 选择 `Guile` / `Pierre` / `Nikki`、`Branch 3` 是否救 `Kid`。
- 剧透提示：本攻略按完成度路线编写，默认包含大量剧情、地点、角色和结局剧透。

## 周目建议

- 第一周目可以按普通路线游玩，但应尽量收齐当前分支限定角色和 `Branch 3` 对应 `Frame`。
- 第二周目建议在 `Harle` 仍在队伍时取得一个早期结局。
- 第三周目再按完成路线回收所有角色，并在条件满足后使用 `Chrono Cross` 召回旧周目角色。
