## 东兰特街道到王都地下

1. 艾斯贝尔和苏菲两人从东兰特街道前往联络港。
2. 在港口剧情后返回巴罗尼亚，进入圣堂和王都地下。
3. 地下通道中理查加入。成年后可以推动幼年时推不动的石门。
4. 中途调查巨大骨骸取得发现物。
5. 离开王都地下后前往南巴罗尼亚街道。

## 沃尔布里奇地下遗迹

1. 南巴罗尼亚街道推进到小屋剧情，之后往下进入遗迹。
2. 帕丝可加入。遗迹的机关以绿色浮台和颜色机关为主。
3. 第一次机关选上方，后续双机关选下方可推进主路。
4. Boss 会造成石化，提前准备解除或让雪莉亚负责回复。
5. 战后继续用浮台和撞石机关开路，离开到格雷尔赛德街道。

顺路检查：

- 南巴罗尼亚街道的花、海边闪光点可取得发现物和魔法卡。
- 遗迹内「映像装置」「寡默的石」都要调查。
- 后续会回到沃尔布里奇，先记下未拿宝箱。

## 格雷尔赛德与沃尔布里奇

1. 进入格雷尔赛德后，先看宿屋委托，再补仓库、港口和公爵邸宝箱。
2. 格雷尔赛德街道的山贼事件是支线 NO.22「击退山贼」，可取得帕丝可称号，回宿屋交付。
3. 公爵邸剧情后返回沃尔布里奇地下遗迹，从右侧路线潜入沃尔布里奇。
4. 按顺序启动北桥、南桥和南门。中途 Boss 战对马利克。
5. 剧情后雪莉亚加入，继续推进到北门。

## 巴罗尼亚城夺回战

1. 从南巴罗尼亚街道回王都地下。
2. 推第二个石门后可调查无名墓标。
3. 秘密通道内理查加入。推箱子上行，到城内部。
4. 城内部多次 Boss 战，注意理查的秘奥义。
5. 王座剧情后，马利克归队。之后从港口返回兰特。
6. 兰特防卫战前单独存档，对应「1 分钟击破 理查！」。

## 新的旅程

1. 剧情后苏菲取得称号，从北兰特道前往后山花田，再回兰特。
2. 兰特家内触发母亲、艾斯贝尔、雪莉亚相关剧情。
3. 西兰特道小屋附近战斗后，雪莉亚和帕丝可归队。
4. 回家 1F 右侧房间剧情后，从兰特西港出海。

## 奥尔・雷尤

1. 下船后马利克取得称号。
2. 左侧小孩可取得龟人相关贵重品，船屋内可喂猫。
3. 宿屋有委托，出口附近有发现物。
4. 商店印花可换魔法卡；宝箱密码为龟人相关词。
5. 从出口进入史特拉塔岩石沙漠。

## 赛布尔・伊佐雷

1. 沙漠中调查仙人掌发现物，旁边学者事件会给珍兽图鉴。
2. 进入赛布尔・伊佐雷后，和塔尔卡进行魔法卡小游戏。重制版可按牌数逐步挑战，越晚牌越多越难。
3. 研究室内外有发现物、魔法卡和帕丝可称号事件。
4. 从左侧出口前往史特拉塔大沙漠东。

## 洛克加刚体内

1. 沙漠剧情后被洛克加刚吞下。
2. 体内胃酸会持续扣血，保持回复料理或手动治疗。
3. 调查金色头像、巨大单细胞等发现物。
4. Boss 战后脱出，沿路前往尤・利贝尔提。

成就提示：洛克加刚本体不是本段主线必须击破。全成就的「击破洛克加刚！」建议后期有偷盗和高输出后再处理。

## 尤・利贝尔提

1. 商业区、居住区、政府区都有宝箱、宿屋委托、喂猫和发现物。
2. 到大统领府剧情后，队伍暂时解散。
3. 街上调查大辉石情报，按剧情追踪几个欢乐人。
4. 帕丝可取得称号后，回大统领府，再从西门前往史特拉塔沙漠遗迹。

## 史特拉塔沙漠遗迹

1. 西沙漠中调查砂人和骸骨附近闪光点。
2. 遗迹机关按颜色顺序处理：
   - 第一处：青、黄。
   - 第二处：黄、赤、青。
   - 宝箱路线：赤、青、黄。
   - 主路后段：赤、青、黄。
3. Boss 战前单独存档，对应「1 分钟击破冥王龙！」。
4. 战后大沧海石失去力量，可调查为发现物。
5. 返回尤・利贝尔提，修伯特加入并取得称号。之后可回兰特、巴罗尼亚、格雷尔赛德等地补支线 NO.15、NO.16、NO.20、NO.23、NO.26、NO.30、NO.33、NO.34、NO.57、NO.63。
6. 从北门前往尤・利贝尔提北港，进入下一页。

## 本页收尾

- 赛布尔・伊佐雷魔法卡小游戏尽早分段处理。
- 格雷尔赛德、奥尔・雷尤、尤・利贝尔提的宿屋委托材料建议随时记录。
- 史特拉塔段结束后，队伍进入芬迪尔线，素材和敌人掉落会明显变化。
