## 斗技岛莱奥特峰

1. 从尤・利贝尔提北港乘船到斗技岛。
2. 先调查火焰徽章、宝箱和宿屋委托，右侧红猫可以喂鱼。
3. 和接待员对话，从斗技场第 1 层挑战到第 5 层。
4. 第 5 层后剧情，修伯特取得称号。
5. 和黑衣侦探对话，前往贝拉尼克南港。

斗技岛后续会用于多项成就：支线 NO.35「秘密作战」、NO.37「谁的主意？」、NO.59「黑衣守护者」、斗技场 Master、全委托、找剑事件、最终挑战和未来篇支线。现在只需按主线推进。

## 贝拉尼克

1. 从贝拉尼克南港进入芬迪尔国境地带，沿路调查桃树。
2. 到岔路往上走进入贝拉尼克。进村后先直奔宿屋，避免部分隐藏事件被主线覆盖。
3. 宿屋住宿可触发支线 NO.06「寒冷之夜」。
4. 宿屋右侧舞台房间有发现物和密码宝箱。
5. 村内猫相关事件需要多次给治疗费，完成后雪莉亚取得称号。
6. 主线剧情后，从宿屋左侧洞口进入芬迪尔山岳隧道。

## 芬迪尔山岳隧道到札维特

1. 隧道内走到分岔口，存档点旁有魔法卡，冰树可调查。
2. 出洞到芬迪尔高原，向左前进到札维特联络港。
3. 港口剧情后坐船到札维特，苏菲取得称号。
4. 札维特入城后先处理宿屋委托、土管附近发现物、套娃摆件、魔法卡奖励 NPC。
5. 城内出现异常 NPC 后，依次和武器屋旁、土管附近、宿屋附近 NPC 对话，帕丝可取得称号。斯尼克研究所后回札维特可处理支线 NO.07～NO.10。
6. 从左上出口前往札维特山。

## 安玛露齐亚之里与斯尼克研究所

1. 札维特山上先往安玛露齐亚之里方向走。
2. 山道宝箱怪事件可取得帕丝可称号，回札维特宿屋交付。
3. 安玛露齐亚之里调查巧克力喷泉、机器人床、帕丝可家宝箱和宿屋委托。
4. 英知之藏剧情后前往斯尼克研究所。
5. 研究所内用红箱机关启动装置，多层推进到地下深处。
6. 地下 2F、地下 4F 有发现物，Boss 战后返回札维特。

## 芬迪尔政府塔

1. 札维特城内调查混乱 NPC，取得入馆证。
2. 进入政府塔后马利克取得称号。
3. 39F 到 40F 需要升降机和蒸汽机关。先关闭蒸汽，再按通路逐层推进。
4. 药品分配顺序：
   - 兵士：苹果软糖。
   - 女性兵士：葡萄软糖。
   - 女性上级兵士：苹果软糖。
   - 上级兵士：桃子软糖。
5. 深处剧情后调查卡兹房间，自动脱出。

## 芬迪尔冰山遗迹

1. 从札维特右上出口进入冰海之道，再前往芬迪尔冰山遗迹。
2. 遗迹核心机关是推冰块击破冰柱。
3. 大晶体冰和大红莲石都可调查为发现物。
4. 深处 Boss 战前单独存档，对应「1 分钟击破 卡兹！」。
5. 战后和同伴对话，返回安玛露齐亚之里。

## 世界中心的孤岛

1. 安玛露齐亚之里的英知之藏剧情后，前往札维特港。
2. 到世界中心的孤岛，艾斯贝尔和苏菲取得称号。
3. 存档后 Boss 战，之后进入「守护之路」阶段。

## 海边洞窟与兰特

1. 剧情后苏菲暂时不能使用。先在札维特宿屋住宿。
2. 乘船经奥尔・雷尤、赛布尔・伊佐雷，再到贝拉尼克南港。
3. 经芬迪尔国境地带到海边洞窟，调查明显石门进入剧情。
4. 回兰特后按顺序触发家中剧情：花园、2F 房间、1F 左房间、楼梯壁画、自己房间、花园。
5. 修伯特取得称号。

## 雪埋遗迹

1. 从贝拉尼克出发，经山岳隧道岔路进入未开雪道。
2. 进入雪埋遗迹，先启动中间大门的电源。
3. 左右两侧小门通过箭头通电。右侧、左侧各自通到最后，给中央大门供电。
4. 右侧通电房间和左侧废铁房间各有发现物。
5. Boss 战前单独存档，对应「1 分钟击破坡旅甲！」。
6. 战后准备前往海边洞窟和佛铎拉。

## 泰洛斯阿斯提

1. 海边洞窟剧情后抵达 13 号地区。飞船内可补给。
2. 出飞船后调查秘密基地和悬崖闪光点，进入泰洛斯阿斯提。
3. 城内宿屋、仓库区、记录区都有发现物和委托。
4. 地下 1F 居住区剧情后，前往康乃尔研究设施。

## 康乃尔研究设施

1. 设施内以传送装置和电池机关为主。每下层前尽量带一个充满电的电池。
2. 地下 1F 可触发修伯特称号事件，地下 2F 进入研究室剧情。
3. 地下 3F、地下 4F 继续换电池推进。
4. 地下 4F 存档点后是拉姆达相关 Boss，单独存档，对应「1 分钟击破拉姆达！」。
5. 战后苏菲归队，离开研究所。

## 巴希斯军事基地

1. 从 66 号地区进入基地，按岔路推进到装置。
2. 箱子机关按颜色道路调整。目标是连通绿色道路到存档点。
3. 途中调查阶级章、安玛露齐亚钥匙。后者用于未踏沙漠隐藏区域。
4. 深处 Boss 后全员取得称号。
5. 返回泰洛斯阿斯提，准备进入拉姆达茧。

## 本页收尾

- 斗技岛、札维特、安玛露齐亚之里、泰洛斯阿斯提的委托都牵涉全成就，缺材料时先记录。
- 康乃尔研究设施地下 4F 的 1 分钟 Boss 容易错过，必须保留存档。
- 取得飞行船自由探索后，先补支线再进入最终迷宫会更稳。
